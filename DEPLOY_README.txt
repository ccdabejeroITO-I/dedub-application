Recommended app entry file:

requirements.txt contents:

Basic deployment steps:
1. Upload both files to your GitHub repo.
2. In Streamlit Cloud, select the repo.
3. Set the main file path to: Streamlit_DeDup_App_cloud_cleaner_v3.py
4. Deploy.

If you want a cleaner app name, you can rename the file to app.py before deploy.
